### Possible causes

- The API key is missing.
- The API key is invalid.

### How to fix

- Check if the API key is present in the "Api-Key" request header.
- Check if the API key is valid.
- Go to the [dashboard](https://itz.am/dashboard/api-keys) and try to generate a new API key.

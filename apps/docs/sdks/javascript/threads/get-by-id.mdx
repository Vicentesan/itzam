description: "Get a thread by its ID."
keywords: ["get", "thread", "ID", "JavaScript", "SDK", "retrieve"]
---

## Usage

```typescript
const thread = await itzam.threads.get("thread_1234567890");

console.log(thread.name);
```

### Parameters

| Parameter  | Type     | Required | Description                  |
| ---------- | -------- | -------- | ---------------------------- |
| `threadId` | `string` | **Yes**  | The ID of the thread to get. |

### Return Value

```typescript
type Thread = {
  id: string; // The unique thread ID
  name: string; // The thread name
  lookupKeys: string[]; // The lookup keys if provided
  createdAt: string; // ISO 8601 timestamp
  updatedAt: string; // ISO 8601 timestamp
};
```

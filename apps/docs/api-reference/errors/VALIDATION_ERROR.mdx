### Possible causes

- Some parameters are of the wrong type.
- Some parameters are missing.
- Some parameters are too small.
- Some parameters are too large.
- Some parameters are not in the correct format.

### How to fix

- Check the error message for more details.
- Check the parameters in the request body.
- Check the docs for the expected parameters.

## Overview

Threads allows you to create and manage conversation threads for continuous messaging. Threads are associated with workflows and runs, enabling you to maintain conversation context across multiple AI generations.

For more information, see [Threads](/get-started/threads).

## What are threads?

Threads are a way to create and manage conversation threads for continuous messaging. Threads are associated with workflows and runs, enabling you to maintain conversation context across multiple AI generations.

All API endpoints are authenticated using API Keys. You can create one in the [API
keys section](https://itz.am/dashboard/api-keys) of the Itzam dashboard.

![API Keys](/images/new-api-key.png)

Now, you can use the API key to authenticate your requests.

In the API:

```bash Bash
curl -X POST \
  -H "Api-Key: YOUR_API_KEY" \
  -d '{"input": "Hello, world!", "workflowSlug": "my_great_workflow"}' \
  https://itz.am/api/v1/generate/text
```
